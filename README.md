# 04_react
React.js is a JavaScript library which is help to build best, fast and complex UI (User Interface).

## Command in React 

### for New App with npx
```terminal
npx create-react-app 01basicreact
```

### for Build with npm
```terminal
 npm run build
```

### for New App with vite
```terminal
npm create vite@latest
```

### for NPM install
```terminal
npm i
```

### for NPM Run
```terminal
npm run dev
```
